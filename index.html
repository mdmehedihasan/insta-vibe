Q-1: How does js works?

⮞JavaScript is a client side scripting language and it runs clients machine it means clients web browser.
⮞JavaScript runs on browser and it has responsible engine for that -
⦿Chrome Browser engine: V8
⦿Edge Browser engine: Chakra
